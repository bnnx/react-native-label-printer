# react-native-label-printer

React Native library for printing labels via Bluetooth on Android and iOS, with support for TSPL and CPCL thermal printers.

## Installation


```sh
npm install react-native-label-printer
```


## Usage


```js
import { multiply } from 'react-native-label-printer';

// ...

const result = multiply(3, 7);
```
